Appendix — Quickstart (5-minute)

  1. Clone Cotopia's repo and copy `.env.example` → `.env`. Set RPC_URL and PRIVATE_KEY (use testnet key).
  2. npm i, then npx hardhat compile.
  3. Deploy to testnet with provided script: npx hardhat run scripts/deploy.js --network goerli.
  4. Update `config.js` or `index.html` with deployed contract address and token address (test ERC-20).
  5. Open demo, connect wallet, test approve → buy → start round flows for Forth Connect AI.

Recommended verification steps

  • Verify contract source on block explorer.
  • Run a small manual end-to-end test: purchase 1 play, consumePlay, confirm events and balances.
  • Replace owner with multisig (Gnosis Safe) before production.
